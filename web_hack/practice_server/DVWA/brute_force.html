Brute Force attack

Brute Force attack is uses trial and error to crack passwords, login credentials, and encryption keys. It is simple yet reliable tactic for gaining unauthorized access to individual accounts and organizations' systems and networks. The chance gets extremely smaller if special and numeric characters are used and the length of password gets longer.
There are largely two types of brute force attack :
  1. Alphabetic substitution

    2. Alphabetic substitution simply tries all possible candidates from a to z. (Special and numeric characters can be included)

  2. Dictionary attack

    3. Dictionary attack uses a file which contains a list of candidates. (The file could contain more than hundreds of thousands of candidates)
